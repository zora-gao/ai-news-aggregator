1它能帮你做什么

把 17 个平台 / 7 大信源（Bloomberg Tech、TechCrunch AI、机器之心、AI News、Crunchbase、Meta、Google AI、Product Hunt 等）的内容聚合到一起， 通过 AI 自动改写中文标题、生成中文摘要、打分排序、提取标签。打开网页就能在几分钟内扫完全网最值得关注的 AI 动态——不用再挨个刷各家网站。

2顶部四个 Tab，各看各的
📰 首页（资讯）
综合资讯流，AI 自动按价值打分排序。每条都有 P0~P3 优先级标注、AI Radar 评分、中文摘要和标签。每天打开 5 分钟扫一遍 P0/P1 就够了。
📦 产品
Product Hunt 当日 / 近 7 天热门产品榜。标题统一改写为「产品名：中文功能简述」格式（如 PicOne：macOS 菜单栏图片上传工具），秒懂是干嘛的。
✨ Skills
GitHub 上 AI 工具 / Agent / Prompt 工程类的热门开源项目榜，按 Star 数和近 7 天涨幅排序，附中文描述与 README 摘要。
🧠 Models
主流大模型一览表：上下文长度、输入输出价格、模态（文本/图像/音频/视频）、发布时间。可按上下文 / 价格 / 时间排序，方便选型。
3资讯卡片上每个标记的含义
P0 必看级：行业重大事件、爆款产品发布、关键技术突破。每天最多十几条。
P1 重要：值得关注的产品更新、研究进展、商业动作。
P2 一般：常规资讯，赶时间可跳过。
P3 低优先：边角信息，仅作存档。
⭐ 85 AI Radar 评分：综合价值分（0-100）。≥85 红色高亮，70-84 琥珀色。
✅ 已读 点过原文后会自动标记，下次回来一眼看出什么没看过。
⭐ 收藏 右上角星星点亮即收藏，事后在右上角 ⭐ 入口查看你的收藏列表。

5三步上手
打开网址，先看「首页」P0 / P1 资讯
页面默认按价值打分倒序排列。看红色 P0 + 琥珀 P1 标签的卡片，5 分钟扫完当日重点。
看到感兴趣的，点星星收藏 / 点标题阅读原文
点击卡片任意位置 = 跳转原文（自动标记已读）；点击右上角 ⭐ = 收藏。
按需切到「产品 / Skills / Models」三个垂直频道
关注产品趋势看 Product Hunt 榜；找开源工具看 Skills；选模型看 Models 对比表。
6关于隐私与数据（重要）

💡 你需要知道这几点

  • 不需要登录注册，打开就能用，所有功能立即可用。
  • 你的 收藏 / 阅读记录 仅保存在你自己的浏览器本地（localStorage），不会上传服务器，同事之间互不可见。
  • 换设备 / 换浏览器数据不会自动同步，需要可在 ⭐ 或 🕘 弹窗内导出 JSON 备份。
  • 清浏览器缓存可能丢失数据，建议定期导出。
  • 页面每 2 小时自动重新生成，刷新即获取最新内容（如缓存未更新可 Cmd+Shift+R 强刷）。
